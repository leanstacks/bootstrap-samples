Side Navigation

Side navigation is a vertical navigation component. Unlike a Navbar component which provides a horizontal component containing menu-style navigation, a side navigation component has a vertical orientation and extends from the left or right side of the page.

How it works

By leveraging the Bootstrap grid system, using a few components like Navs and Scrollspy, and some positioning utilities for good measure, you can create a side navigation menu that dynamically updates as the page scrolls.

  • Use the grid system to create side navigation and main content columns
  • Create a vertical nav component in the side bar
  • Add Scrollspy to automatically update the navigation items as the page scrolls
  • Use position: sticky; to hold the navigation menus in place while the main content scrolls

Columns

Using the Bootstrap grid system, create a single row containing two columns. A small column to contain the side navigation, and a larger column to contain the page content. 

  
    <div class="row">
      <div class="col-3 bg-primary text-white cs-sidenav"> <!-- Side -->
        <h3 class="mt-3">Topics</h3>
        <nav class="nav flex-column" id="side-nav">
          <a class="nav-link h5 mb-0" href="/">Side Navigation</a>
          <a class="nav-link" href="#topic-how-it-works">How it works</a>
          ...
        </nav>
      </div>

      <div class="col-9 cs-content"> <!-- Content -->
        <h2 id="topic-how-it-works" class="mb-3">How it works</h2>
        <p>...</p>
        ...
      </div>
    </div>

Vertical Navigation

To create a side navigation menu, use a Bootstrap Nav component. Apply the flex-column CSS class to make the navigation items stack vertically.

      <a class="nav-link h5 mb-0" href="/">Side Navigation</a>
      <a class="nav-link" href="#topic-how-it-works">How it works</a>

Scroll Spy

Use Scrollspy to automatically update the navigation menu to reflect the scroll position in the main content area. Because we have a Navbar at the top of the page which uses position: sticky;, offset the detection point for Scrollspy using data-offset="80". 

  
    <body data-spy="scroll" data-target="#side-nav" data-offset="80"> 
    ...
    </body>

Sticky Positioning

In this example position: sticky; is applied to the Navbar component at the top of the page and the Nav component in the side bar. Sticky positioning holds an element at the top of the page once a user scrolls past it. In this example, both the Navbar and the Nav are located at the top of the page, so sticky positioning holds the menus in place allowing the user to scroll the main content.

You may use the .sticky-top class provided with Bootstrap's Position utilities; however, this example applies sticky positioning though a couple of custom CSS classes because some additional styling is required for the Navbar and side Nav components also (specifically height and z-index). 

  
    .cs-navbar {
      min-height: 4rem;
      position: sticky;
      top: 0;
      z-index: 1100;
    }
    
    .cs-sidenav {
      position: sticky;
      top: 4rem;
      height: calc(100vh - 4rem);
      z-index: 1000;
    }
